# Updates & Changelog

This folder contains all update and changelog documentation for Proxmox Monitor.

## Files

### Release Notes
- `RELEASE_NOTES.md` - Version releases and features

### Feature Updates
- Add feature update files here as: `update-YYYY-MM-DD-feature-name.md`

### Format

Each update file should follow this structure:

```markdown
# Update: [Feature Name]
**Date:** YYYY-MM-DD
**Version:** X.Y.Z
**Branch:** feature/xxx or develop

## Summary
Brief description of changes

## Changes
- Change 1
- Change 2

## Files Modified
- `file1.py`
- `file2.yaml`

## How to Use
Steps to test or implement

## Related Issues
Links to issues or PRs
```

## Examples

- `update-2026-05-14-first-start-setup.md` - First-start setup with inline buttons
- `update-2026-05-14-docker-support.md` - Docker containerization
